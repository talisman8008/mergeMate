# FirstMerge: The Recruitment Evolution (B2B SaaS)

## The Vision
FirstMerge is uniquely positioned to bridge the gap between **Open Source Contributors** and **Technical Recruiters**. While LinkedIn relies on self-reported skills (which are often exaggerated) and GitHub is too dense for non-technical recruiters to parse, FirstMerge translates raw GitHub data into clear, verified performance signals. 

By adding a Recruiter Portal, FirstMerge evolves from a developer utility into a highly monetizable B2B recruitment marketplace.

---

## 🎯 Core Features for the Roadmap

### 1. The "Open to Work" Signal
- **For Developers:** A simple toggle in their dashboard: *"I am open to new opportunities."*
- Developers can specify their preferred role types (Frontend, Backend, DevRel), salary expectations, and timezone.
- This creates an exclusive pool of active, high-quality candidates.

### 2. Recruiter Dashboard & Advanced Search
- **For Companies:** A dedicated B2B portal where recruiters can query the FirstMerge database using verified data.
- **Example Searches:**
  - *"Find me developers who have merged at least 5 PRs in React repositories with >1000 stars."*
  - *"Show me developers with a 'Friendliness Score' over 90% and experience in Python."*
- Filters based on actual, cryptographically verified code contributions rather than LinkedIn keyword stuffing.

### 3. Verified Skill Badges (The "Proof of Work" Protocol)
- Recruiters can instantly see a developer's FirstMerge Badges (e.g., *The Machine: 500+ Commits*, *OS Contributor*).
- This acts as an automated technical screen. If a candidate has a "React Master" badge backed by 50 merged PRs, the recruiter knows they can actually code.

### 4. Direct Outreach & Matchmaking
- Recruiters can use "Merge Credits" (a paid token system) to message developers directly through the platform.
- Messages have a higher response rate because developers know they were selected based on their actual code, not an automated LinkedIn scraper.

---

## 💰 Monetization Strategy (B2B SaaS)
- **Developers:** Always 100% Free.
- **Recruiters (Standard):** $99/month for access to the candidate database and basic search filters.
- **Recruiters (Enterprise):** $499/month for advanced API access, unlimited outreach, and automated candidate matchmaking based on company tech stacks.

## 🚀 Why This Wins
Technical recruiting is currently broken. Companies spend thousands of dollars on LeetCode-style interviews because they can't trust resumes. FirstMerge solves this by providing **Proof of Work**. A recruiter using FirstMerge isn't guessing if a candidate is good; the candidate's verified PRs and Friendliness Score have already done the screening for them.
